1 - check argc
2 - malloc data struct + malloc protection + init data struct
3 - start_server (socket, bind, listen)
4 - main_loop

main_loop:
1 - set up select (fd_set set_fds)
2 - start an infinite loop
3 - FD_ZERO(&set_fds) - clears set every loop
4 - FD_SET(server_socket, &set_fds) - add server socket to the set
5 - Add all the client fds to the set in a for loop
6 - SELECT to flag the fds ready to be read
7 - FD_ISSET to see which fds are ready
    7.1 - Check if the server fd is ready
        7.1.1 - If yes accept connection
        7.1.1 - And add_client()
    7.2 - Check if other fd is ready
        7.2.1 - If yes recv()
            7.2.1.1 - If recv returns <= 0 remove_client
            7.2.1.1 - Else read from that fd till the entire message is extracted (use extract_message())
